# Bosch_BME680_Jetson

Aligned for Nvidia Jetson Xavier with I2C Bus #8.
I2C connections as per Jim's tutorial on Jetsonhacks.com.


How to compile
---

In the directory:
`make`

How to execute
---

`./main`
